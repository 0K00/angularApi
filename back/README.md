## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Task

### Example Task Management API

Available Endpoints:
List of Tasks:

Endpoint: GET /tasks
Description: Retrieve the list of all tasks.
Expected Response:
```json
[
  {
    "id": 1,
    "title": "Grocery Shopping",
    "description": "Buy fruits and vegetables",
    "completed": false
  },
  {
    "id": 2,
    "title": "Study for Exam",
    "description": "Chapters 1 to 5",
    "completed": true
  },
  // ... other tasks
]
````

Task Details:

Endpoint: GET /tasks/{id}
Description: Retrieve details of a specific task.
```json
{
  "id": 1,
  "title": "Grocery Shopping",
  "description": "Buy fruits and vegetables",
  "completed": false
}
````

Create a Task:

Endpoint: POST /tasks
Description: Create a new task.
Request Body:
```json
{
  "title": "New Task",
  "description": "Description of the new task"
}
````

Expected Response:
```json
{
  "id": 3,
  "title": "New Task",
  "description": "Description of the new task",
  "completed": false
}
````

Update a Task:

Endpoint: PUT /tasks/{id}
Description: Update details of an existing task.
Request Body:
```json
{
  "title": "Updated Task",
  "description": "New description of the task",
  "completed": true
}
````

Expected Response:
```json
{
  "id": 3,
  "title": "Updated Task",
  "description": "New description of the task",
  "completed": true
}
```

Delete a Task:

Endpoint: DELETE /tasks/{id}
Description: Delete a specific task.
